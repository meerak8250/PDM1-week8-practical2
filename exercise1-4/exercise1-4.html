Instructions

sketch.js contains almost complete code for a scene generator. It allows the user to place sprites on a drawing areas to design a pixel-art scene that can be used in the solution to the previous exercise. Run the code and try creating a scene.

The only thing missing is the save functionality, which should allow the user to download their creation as a text file. Complete the saveLevel() function to add this functionality. The images and their locations are stored in the placedSprites global array (as Sprite objects). The information in the file should have exactly the same format as the text files you worked with in the previous exercise.

Read the existing code before starting to write your own! You should only add / edit code inside the saveLevel() function.

You should be able to use the downloaded text file in your solution to exercise 1-3. To do this, move the downloaded file into the assets folder and edit the name of the file loaded into the sketch with loadStrings().

This exercise doesn't have a check file because the output of the saveLevel() function can't be automatically tested.

You only need to add code inside the saveLevel() function.

You will need a for loop to iterate through each item in the placedSprites global array, which contains all Sprite object placed on the drawing area. The Sprite class, defined at the bottom of sketch.js, is exactly the same as the Sprite class in the previous exercise.

Inside the for loop, you will need to use the information contained in the Sprite object to build a line of text that will be saved for the file. The hardest part is to get the name of the Sprite. You will need a conditional and the Sprite method, getImage(). Use the conditional to figure out which of the global image variables the sprite is equal to e.g.

sprite.getImage() === grass

...the above boolean expression checks if the image in the Sprite is the same as the image store in the grass variable.

You will also need to use the Sprite methods getX() and getY(). To convert these three pieces of information into a comma separated string, use a string concatenation method. We covered string concatenation in week 6 lecture 1.
